在您要接入的网站内添加嵌入如下的代码：

<script src="{$domain}/assets/layer/ai_service_diy_{$business_id}.js" charset="UTF-8"></script>

则会在您访问所添加的网站，自动弹出客服聊天窗口！


  • 如果你们的网站有自己的 用户id 、用户名、头像信息，您可以通过如下方式传入
<script type="text/javascript">   var ai_service={};   ai_service.visiter_id='';//访客id   ai_service.visiter_name='';//访客昵称   ai_service.avatar='';//访客头像绝对路径 </script>
参数 解释
visiter_id 访客的唯一标识，可以是你网站的用户id 或者 其他标识，不传系统自动生成 uuid
visiter_name 访客的昵称，不传的话，系统自动会拼接 '访客' + uid
avatar 访客的头像，即您网站的用户头像, 不传默认是系统的通用头像

你可以在任何位置添加以下客服聊天链接，实现接入智能链接访问模式：

{$domain}/index/index/home?visiter_id=&visiter_name=&avatar=&groupid=0&business_id={$business_id}
参数 解释
visiter_id 访客的唯一标识，可以是你网站的用户id 或者 其他标识，不传系统自动生成 uuid
visiter_name 访客的昵称，不传的话，系统自动会拼接 '访客' + uid
avatar 访客的头像，即您网站的用户头像, 不传默认是系统的通用头像
groupid 客服分组ID
business_id 系统固定参数，不可修改

你可以在任何浏览器通过扫码，实现接入二维码访问模式：

{foreach $service as $k=>$v} {/foreach}
客服名称 访问连接 访问连接二维码 微信链接 微信链接二维码
{$v['user_name']} {$v['personal']}
{$v['personalwechat']}
